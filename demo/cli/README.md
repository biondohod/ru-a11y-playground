# CLI demo data

This folder contains URL lists and generated reports for `ru-a11y` CLI demos.

- `urls.bad.txt` - pages with intentional accessibility issues
- `urls.good.txt` - pages with improved accessibility
- `urls.mixed.txt` - mixed pages for showcase runs
- `urls.all-routes.txt` - full route list from `src/App.jsx`
- `reports/` - generated JSON, Markdown, and styled HTML outputs
